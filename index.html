Iniciação Científica: Predição de Vazão de Petróleo

Deep Learning // UNICAMP // Pesquisa Acadêmica

Nesta pesquisa orientada pelo Prof. Dr. João Roberto Bertini Junior, enfrento o desafio da escassez de dados em campos de petróleo recém-descobertos. Utilizo técnicas de Domain Adaptation e Transfer Learning para transpor o conhecimento de campos maduros para novos poços.


O coração técnico do projeto envolve a implementação de arquiteturas híbridas CNN-LSTM. As camadas convolucionais extraem padrões morfológicos dos sinais geofísicos, enquanto as camadas LSTM tratam a dependência temporal da vazão. O objetivo é reduzir drasticamente o erro de predição em cenários de dados limitados (Small Data).

Analista de Documentos IA (PDF Summarizer)

NLP // Generative AI // Streamlit

Desenvolvi uma aplicação end-to-end que resolve a fadiga de leitura de documentos técnicos longos. O sistema utiliza a API do Google Gemini 2.5 Flash para realizar processamento de linguagem natural em tempo real sobre arquivos PDF.


Diferente de wrappers simples, este projeto foca em Engenharia de Prompt (com modos de Resumo, Análise Técnica e Tradução) e segurança de infraestrutura. Implementei travas de backend para controlar o consumo de tokens e limites de upload, garantindo a sustentabilidade financeira e técnica da ferramenta.
