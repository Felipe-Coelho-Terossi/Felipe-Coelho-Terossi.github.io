API REST de Autenticação com JWT

Node.js · TypeScript · PostgreSQL · React

API de autenticação production-ready implementando o padrão de dois tokens JWT: access token de curta duração (15 min) verificado sem banco de dados, e refresh token persistido para renovação automática de sessão.

Arquitetura em camadas (routes → controllers → services → utils), middleware de proteção de rotas, hash de senhas com bcrypt e frontend React com interceptors Axios para renovação transparente de tokens.

Todo API — Docker & CI/CD

Node.js · PostgreSQL · Docker · GitHub Actions

API REST de tarefas completamente containerizada com Docker e pipeline de CI/CD automático via GitHub Actions. A cada push na branch main, os testes rodam automaticamente com banco real e a imagem é publicada no Docker Hub.

Banco de dados PostgreSQL persistido via Docker Volume, healthcheck garantindo ordem de inicialização dos containers, e deploy automático na nuvem via Render + Neon.

IC: Predição de Vazão de Petróleo

Deep Learning · UNICAMP · Transfer Learning
